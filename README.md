# Notes API

A simple and efficient RESTful API for managing notes, built using **Node.js** and **Express.js**.  
This project demonstrates CRUD operations and serves as a foundation for learning backend development.

## Features
- Get all notes
- Get single note
- Add note
- Update note
- Delete note

## Tech Stack
- Node.js
- Express.js

## Installation & Run
```bash
npm install
node notes.js
